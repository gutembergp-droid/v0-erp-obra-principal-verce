5. FLUXOS DE GOVERNANCA IMPLEMENTADOS
--------------------------------------------------------------------------------

5.1 Efetivacao de Colaborador
    Pre-Cadastro -> Vinculo -> Documentacao -> SST -> Aprovacao -> Efetivacao
    
    Regras:
    - Documentacao obrigatoria deve estar OK
    - ASO admissional deve estar valido
    - NRs obrigatorias devem estar OK
    - Aprovacoes hierarquicas concluidas
    - Somente apos checklist completo libera efetivacao

5.2 Consolidacao de Mao de Obra
    RH -> Custos -> Gerencia -> Financeiro
    
    Regras:
    - RH consolida mas NAO paga e NAO faz rateio
    - Checklist de verificacoes obrigatorio
    - Observacao obrigatoria na consolidacao
    - Rastreabilidade completa

5.3 Convencoes Coletivas
    Criacao -> Revisao -> Aprovacao -> Publicacao
    
    Regras:
    - Versao ativa nao pode ser editada
    - Nova versao gera historico
    - Aprovacao hierarquica obrigatoria

5.4 Folha de Pagamento
    Consolidacao -> RH Aprova -> Custos Rateia -> Gerencia Aprova -> Financeiro Paga
    
    Regras:
    - Fluxo sequencial obrigatorio
    - Cada etapa bloqueia anterior
    - Auditoria em cada transicao
